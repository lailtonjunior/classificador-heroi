# Classificador de Nível de Herói em Python
Este projeto contém um pequeno script em Python que classifica o nível de um herói com base em sua quantidade de experiência (XP).
Ele utiliza uma estrutura de decisão para determinar o nível correspondente de acordo com a tabela proposta.

## Objetivo
Receber o nome e a quantidade de XP de um herói e, com base nisso, exibir uma mensagem indicando seu nível.
